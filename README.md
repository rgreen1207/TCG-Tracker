# 🃏 Pokemon Price Tracker

A self-hosted Raspberry Pi price tracker for Japanese (and English) Pokémon cards, booster packs, booster boxes, and special sets. Runs 24/7, monitors multiple price sources, sends push notifications to your phone when deals are found, and tracks your personal collection with graded card support.

---

## Features

- **Price tracking** across multiple sources — TCGdex (always on), eBay, PriceCharting, and pokemontcg.io; active sources shown on the dashboard
- **No China/HK sellers** — automatically excluded at the API level
- **MSRP comparison** — every price shown with ▲/▼ vs MSRP
- **Both ¥ and $** displayed everywhere, with live exchange rate
- **Fuzzy search** — type "charmader" and get "Charmander"; EN names map to JP automatically
- **Booster packs, boxes, special boxes** — dedicated pages with filter tabs
- **Collection tracking** — raw, PSA-graded, TAG-graded; portfolio total value
- **Watchlist** — set a target price per item, get notified when it's hit
- **Push notifications** via [Pushover](https://pushover.net) (toggle on/off in UI)
- **Sparkline charts** — 30-day price history per product
- **Admin panel** — manage sets, products, cards; protected by JWT login
- **Export** — CSV and JSON collection export
- **Public read / admin write** — collection and watchlist are public; editing requires login
- **Mobile-first** responsive dark UI
- **HTTPS** via Nginx + Let's Encrypt / self-signed cert
- **systemd** service — auto-starts on boot, restarts on crash

## Quick Start (Raspberry Pi)

Works on any Pi, any username (`pi`, `ryan`, `ubuntu` — doesn't matter).

```bash
# 1. Clone the repo directly on your Pi
git clone https://github.com/rgreen1207/TCG-Tracker
cd TCG-Tracker

# 2. Run the one-command setup script
bash deploy/setup.sh
```

The script auto-detects your username and repo location. When it finishes, it prints a full step-by-step guide covering credentials, service start, port-forwarding rules (with a table showing your Pi's exact local IP), and optional free domain setup.

```bash
# 3. Follow the printed instructions — starting with:
nano /home/<your-user>/TCG-Tracker/.env   # path printed by setup.sh

# 4. Start the service
sudo systemctl start tcg-tracker

# 5. Open in browser
# Local network:  https://<pi-ip>          (printed by setup.sh)
# From internet:  https://<public-ip>      (after port-forwarding — see below)
```

To deploy on any other Pi in future, the same two commands work unchanged:
```bash
git clone https://github.com/rgreen1207/TCG-Tracker && cd TCG-Tracker && bash deploy/setup.sh
```

## Configuration (`.env`)

**The app runs with zero credentials configured.** Features enable progressively as you add keys. The dashboard shows which sources are currently active.

Copy `.env.example` → `.env` and fill in what you need:

### Always required (security)

| Variable | Description |
|---|---|
| `SECRET_KEY` | Random 32-byte hex string — auto-generated by `setup.sh` |
| `ADMIN_PASSWORD` | Password for the admin panel login |

### Optional — enables eBay live listings

| Variable | Description |
|---|---|
| `EBAY_APP_ID` | eBay developer App ID |
| `EBAY_CLIENT_SECRET` | eBay client secret |

Without eBay credentials the app still shows card data and market prices from TCGdex (TCGPlayer/CardMarket aggregated). eBay adds live Buy-It-Now listings with direct purchase links and real-time seller prices.

### Optional — enables PriceCharting market data

| Variable | Description |
|---|---|
| `PRICECHARTING_API_KEY` | PriceCharting API key |

Enables historical market averages for sealed products and graded cards (PSA 10, PSA 9, TAG grades). Requires a **paid** PriceCharting subscription. Without it, graded pricing falls back to TCGdex and eBay.

### Optional — enables pokemontcg.io enrichment

| Variable | Description |
|---|---|
| `POKEMONTCG_API_KEY` | pokemontcg.io API key |

Enables richer English card metadata and TCGPlayer price history via pokemontcg.io. Free key, but a key is required — the keyless rate limit (1,000 req/day) is too low for background polling. Note: **English sets only** — Japanese cards always use TCGdex regardless of this setting.

### Optional — enables push notifications

| Variable | Description |
|---|---|
| `PUSHOVER_USER_KEY` | Your Pushover user key |
| `PUSHOVER_API_TOKEN` | Pushover app token |

Without Pushover, watchlist alerts are still logged in the UI — you just won't get a phone notification.

### Always-on sources (no configuration needed)

| Source | What it provides |
|---|---|
| **TCGdex** (`api.tcgdex.net`) | Card metadata, images, EN+JP names, TCGPlayer & CardMarket prices |
| **Frankfurter** (`frankfurter.app`) | Live USD/JPY/EUR exchange rates |

## API Keys — Where to Get Them

### eBay Browse API (optional — enables live listings)
1. Go to **[developer.ebay.com](https://developer.ebay.com)** and create a free account
2. My Apps → Create App → set name and environment (Production)
3. Enable the **Browse API** in your app settings
4. Copy **App ID (Client ID)** → `EBAY_APP_ID` and **Client Secret** → `EBAY_CLIENT_SECRET`

*What you get:* Active Buy-It-Now listings with direct links. China/HK sellers are automatically excluded.

### Pushover (optional — enables phone push alerts)
1. Sign up at **[pushover.net](https://pushover.net)** — free account
2. Note your **User Key** on the dashboard → `PUSHOVER_USER_KEY`
3. Create an application at **[pushover.net/apps/build](https://pushover.net/apps/build)** → `PUSHOVER_API_TOKEN`
4. Install the Pushover app on your phone (one-time $5 licence, iOS/Android)

*What you get:* Push notification when a watchlist item drops to your target price.

### PriceCharting (optional — enables graded and sealed market history)

Requires a paid subscription at **[pricecharting.com/api](https://www.pricecharting.com/api)**.

1. Subscribe and log in at pricecharting.com
2. Go to your account API settings and generate a key
3. Add it to `.env` as `PRICECHARTING_API_KEY`

*What you get:* Historical market averages for sealed products and graded card prices broken down by grade (PSA 10, PSA 9, TAG 10, etc.).

### pokemontcg.io (optional — enriches English card data)

Free key at **[pokemontcg.io](https://pokemontcg.io)**.

1. Register at pokemontcg.io and confirm your email
2. Copy your API key from the dashboard
3. Add it to `.env` as `POKEMONTCG_API_KEY`

*What you get:* Richer English card metadata and TCGPlayer market prices as an additional price source alongside TCGdex. **Does not cover Japanese cards** — TCGdex handles those regardless.

### No-key sources (always active)

**TCGdex** (`api.tcgdex.net`) — completely free, no signup required.
Provides card metadata in 10+ languages including Japanese, card images,
and market prices sourced from TCGPlayer (USD) and CardMarket (EUR).

**Frankfurter** (`frankfurter.app`) — completely free, no signup required.
Provides live USD/JPY/EUR exchange rates, refreshed daily.

## Router Port Forwarding

`setup.sh` prints a ready-to-use table with your Pi's actual local IP. Here's the full process:

### 1 — Find your Pi's local IP

```bash
hostname -I   # e.g. 10.0.0.141
```

### 2 — Log into your router

Open a browser on any device on your home network and go to:
- `http://192.168.1.1` (most common)
- `http://192.168.0.1` (some routers)
- `http://10.0.0.1` (some ISP-provided routers)

Default admin credentials are usually printed on a sticker on the router. If you've never changed them, try `admin` / `admin` or `admin` / `password`.

### 3 — Add port forwarding rules

Look for a section called **NAT**, **Port Forwarding**, **Virtual Servers**, or **Applications & Gaming** (name varies by manufacturer).

Add these two rules (replace `<Pi's local IP>` with the IP from step 1):

| External Port | Internal IP | Internal Port | Protocol |
|---|---|---|---|
| 443 | `<Pi's local IP>` | 443 | TCP |
| 80 | `<Pi's local IP>` | 80 | TCP |

### 4 — Give your Pi a static local IP

Without this, your Pi's local IP can change after a reboot and break the forwarding rules.

In your router look for **DHCP Reservations**, **Static Lease**, or **Address Reservation**. Find your Pi's MAC address (run `ip link show` on the Pi) and bind it to the IP you used above.

### 5 — Router-specific guides

**[portforward.com](https://portforward.com)** has step-by-step guides for virtually every consumer router, searchable by brand and model number.

## Free HTTPS with Let's Encrypt

Connecting over HTTPS with a self-signed cert (what `setup.sh` installs by default) will show a browser security warning. To remove it and get a real trusted cert, you need a domain name pointing at your public IP.

### Option A — Free dynamic DNS subdomain (easiest)

If you don't own a domain, these services give you a free subdomain that stays pointed at your home IP even when it changes:

- **[DuckDNS](https://www.duckdns.org)** — free, simple, has a dedicated Raspberry Pi setup guide
- **[No-IP](https://www.noip.com)** — free tier available, widely used

Both services provide a small daemon you run on the Pi that updates your subdomain whenever your public IP changes.

### Option B — Use a domain you already own

Point an `A` record at your public IP in your domain registrar's DNS settings.

### Get the certificate

Once your domain resolves to your Pi's public IP:

```bash
sudo certbot --nginx -d yourdomain.com
```

Certbot automatically edits the Nginx config to use the new cert. The cron job installed by `setup.sh` renews it every 90 days silently.

## Service Management

```bash
# Start / stop / restart
sudo systemctl start tcg-tracker
sudo systemctl stop tcg-tracker
sudo systemctl restart tcg-tracker

# View live logs
sudo journalctl -u tcg-tracker -f

# Check status
sudo systemctl status tcg-tracker
```

> The service is named `tcg-tracker`. The file `deploy/pokemon-tracker.service` in the repo is a documentation template only — `setup.sh` writes the actual service file to `/etc/systemd/system/tcg-tracker.service` dynamically with your machine's real username and paths baked in.

## Adding Cards & Products

1. Log in at `/login` with your admin password
2. Go to **Admin Panel** (`/admin`)
3. Add a **Set** first (e.g. "Scarlet & Violet — Battle Partners")
4. Add **Products** (booster pack, booster box, ETB, etc.) linked to the set
5. Add **Cards** (individual cards with EN/JP names, card numbers, rarity)
6. The price poller will automatically start tracking prices at the next poll interval
7. To poll immediately: click **⟳ Poll Now** in the top bar

## English → Japanese Search

The search works as follows:
1. You type an English name (even with typos)
2. **Fuse.js** (client-side) does instant fuzzy matching against the full card index
3. If fewer than 3 results, **rapidfuzz** (server-side Python) augments with deeper matching
4. Results show both English and Japanese names
5. The Japanese name is used when querying eBay for JP-market listings

## Security Notes

- Admin password is stored as a **bcrypt hash** — never in plaintext
- Login is rate-limited to **5 attempts / minute** at Nginx, with a **15-minute lockout** after 5 failures
- All cookies are `HttpOnly; SameSite=Strict` (add `Secure` flag in nginx for production)
- Nginx blocks direct access to `.env` and `data/` paths
- Security headers (HSTS, X-Frame-Options, CSP) set by Nginx
- The SQLite database is **not** exposed via any API endpoint

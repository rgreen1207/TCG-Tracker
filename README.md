# 🃏 Pokemon Price Tracker

A self-hosted Raspberry Pi price tracker for Japanese (and English) Pokémon cards, booster packs, booster boxes, and special sets. Runs 24/7, monitors multiple price sources, sends push notifications to your phone when deals are found, and tracks your personal collection with graded card support.

---

## Features

- **Price tracking** across eBay (active BIN only), PriceCharting, and Frankfurter FX rates
- **No China/HK sellers** — automatically excluded at the API level
- **MSRP comparison** — every price shown with ▲/▼ vs MSRP
- **Both ¥ and $** displayed everywhere, with live exchange rate
- **Fuzzy search** — type "charmader" and get "Charmander"; EN names map to JP automatically
- **Booster packs, boxes, special boxes** — dedicated pages with filter tabs
- **Collection tracking** — raw, PSA-graded, TAG-graded; portfolio total value
- **Watchlist** — set a target price per item, get notified when it's hit
- **Push notifications** via [Pushover](https://pushover.net) (toggle on/off in UI)
- **Sparkline charts** — 30-day price history per product
- **Admin panel** — manage sets, products, cards; protected by JWT login
- **Export** — CSV and JSON collection export
- **Public read / admin write** — collection and watchlist are public; editing requires login
- **Mobile-first** responsive dark UI
- **HTTPS** via Nginx + Let's Encrypt / self-signed cert
- **systemd** service — auto-starts on boot, restarts on crash

## Quick Start (Raspberry Pi)

```bash
# 1. Copy the project to your Pi
scp -r pokemon-tracker pi@raspberrypi.local:~/

# 2. Run the one-command setup script
cd ~/pokemon-tracker
bash deploy/setup.sh

# 3. Edit your credentials
nano .env

# 4. Start the service
sudo systemctl start pokemon-tracker

# 5. Open in browser
# Local: https://<pi-ip>
# After port-forwarding 443 on your router: https://<your-public-ip>
```

## Configuration (`.env`)

Copy `.env.example` → `.env` and fill in:

| Variable | Description |
|---|---|
| `SECRET_KEY` | Random 32-byte hex string (auto-generated by setup.sh) |
| `ADMIN_PASSWORD` | Your admin login password |
| `EBAY_APP_ID` | eBay developer App ID — [developer.ebay.com](https://developer.ebay.com) |
| `EBAY_CLIENT_SECRET` | eBay client secret |
| `PRICECHARTING_API_KEY` | Free at [pricecharting.com/api](https://www.pricecharting.com/api) |
| `POKEMONTCG_API_KEY` | Free at [pokemontcg.io](https://pokemontcg.io) (optional but gets higher rate limits) |
| `PUSHOVER_USER_KEY` | Your Pushover user key — [pushover.net](https://pushover.net) |
| `PUSHOVER_API_TOKEN` | Pushover app token — [pushover.net/apps/build](https://pushover.net/apps/build) |
| `POLL_INTERVAL_SECONDS` | How often to poll prices (default: 21600 = 6 hours) |
| `NOTIFY_COOLDOWN_HOURS` | Min hours between alerts for the same item (default: 12) |

## API Keys — Where to Get Them

### eBay Browse API (required for live listings)
1. Go to [developer.ebay.com](https://developer.ebay.com)
2. Create an account → My Apps → Create App
3. Copy **App ID (Client ID)** and **Client Secret**
4. Enable the **Browse API** in your app settings

### PriceCharting (market averages)
1. Visit [pricecharting.com/api](https://www.pricecharting.com/api)
2. Sign up for a free API key
3. Free tier covers what we need

### Pushover (push notifications)
1. Sign up at [pushover.net](https://pushover.net) — free account
2. Install the Pushover app on your phone (one-time $5 licence)
3. Note your **User Key** from the dashboard
4. Create an application at [pushover.net/apps/build](https://pushover.net/apps/build) — note the **API Token**

### Pokemon TCG API (metadata)
1. Visit [pokemontcg.io](https://pokemontcg.io) → Get API Key
2. Free, no credit card required

## Adding Cards & Products

1. Log in at `/login` with your admin password
2. Go to **Admin Panel** (`/admin`)
3. Add a **Set** first (e.g. "Scarlet & Violet — Battle Partners")
4. Add **Products** (booster pack, booster box, ETB, etc.) linked to the set
5. Add **Cards** (individual cards with EN/JP names, card numbers, rarity)
6. The price poller will automatically start tracking prices at the next poll interval
7. To poll immediately: click **⟳ Poll Now** in the top bar

## Collection Grading

Supported graders:

| Grader | Notes |
|---|---|
| **PSA** | Full grade scale 1–10. PriceCharting has PSA price data for popular cards. |
| **TAG** | Technical Authentication & Grading (UK). eBay sold listing prices used as reference. |

When adding to your collection:
- Select **Graded** status
- Choose grader (PSA or TAG)
- Enter grade (e.g. 10, 9.5, 9)
- The tracker will look up the graded price specifically

## English → Japanese Search

The search works as follows:
1. You type an English name (even with typos)
2. **Fuse.js** (client-side) does instant fuzzy matching against the full card index
3. If fewer than 3 results, **rapidfuzz** (server-side Python) augments with deeper matching
4. Results show both English and Japanese names
5. The Japanese name is used when querying eBay for JP-market listings

## Database

SQLite file at `data/tracker.db`. Key tables:

| Table | Purpose |
|---|---|
| `sets` | Pokemon set metadata |
| `products` | Booster packs, boxes, special sets |
| `cards` | Individual cards with EN/JP names |
| `price_history` | Every price point recorded (timestamped) |
| `collection` | Your owned cards/products with grade info |
| `watchlist` | Items you're watching with target prices |
| `alert_log` | Every push notification sent |
| `notification_config` | Pushover credentials + enabled flag |
| `fx_rates` | Cached USD/JPY exchange rates |
| `login_attempts` | Brute-force protection |

Backup: `cp data/tracker.db data/tracker.db.bak`

## Security Notes

- Admin password is stored as a **bcrypt hash** — never in plaintext
- Login is rate-limited to **5 attempts / minute** at Nginx, with a **15-minute lockout** after 5 failures
- All cookies are `HttpOnly; SameSite=Strict` (add `Secure` flag in nginx for production)
- Nginx blocks direct access to `.env` and `data/` paths
- Security headers (HSTS, X-Frame-Options, CSP) set by Nginx
- The SQLite database is **not** exposed via any API endpoint
